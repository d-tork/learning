# Git

## Commit message conventions

| commit message types |  | scope examples |
|----------------------|--|----------------|
| feat                 |  | init           |
| fix                  |  | runner         |
| docs                 |  | watcher        |
| style                |  | config         |
| refactor             |  | (etc.)         |
| test                 |  |                |
| chore                |  |                |
| build                |  |                |

> <type> (<scope>): <Subject>

as in 

> feat (config): Add min and max as config options.
